# 《夏蝉的诺言》Cicada Promise

## 项目概要
- **类型**: 社会派悬疑 + 叙述性诡计
- **核心**: 双重时间轴（A线2026 / B线1996），最后揭示B线是30年前
- **主题**: 极致的守护、时间的复仇、被遗忘的青春悲剧

## 章节结构

### 第一幕：谜题与铺垫（第1-6章）
- A线（2026）: 刑侦 - 张扬密室死亡
- B线（1996）: 青春 - 林默与夏晓晓的相遇

### 第二幕：深入与危机（第7-14章）
- A线（2026）: 调查展开，寻找夏晓晓
- B线（1996）: 霸凌升级，悲剧发生

### 第三幕：汇合与揭示（第15-20章）
- A线（2026）: 周睿揭示真相
- B线（1996）: 闪回 - 30年的复仇计划
